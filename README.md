# Machine Learning basic algorithms with plain Python
![status](https://img.shields.io/badge/status-up-brightgreen)

This repo contains basic (classic) algorithms used in machine learning. All implementations are built from scratch and without using additional Python libraries.

* [Linear Regression](https://nbviewer.org/github/landisland/MachineLearningAlgorithmsWithPlainPython/blob/master/linear%20regression/linear_regression_from_scratch.ipynb)
* [Logistic Regression](https://nbviewer.org/github/landisland/MachineLearningAlgorithmsWithPlainPython/blob/master/logistic%20regression/logistic_regression_from_scratch.ipynb)
* [Perceptron (with clear step by step plots!)](https://nbviewer.org/github/landisland/MachineLearningAlgorithmsWithPlainPython/blob/master/perceptron/perceptron_from_scratch.ipynb)
* [Decision Trees (regression tree and classification tree)](https://nbviewer.org/github/landisland/MachineLearningAlgorithmsWithPlainPython/blob/master/decision%20tree/decision_tree_from_scratch.ipynb)
* [SVM](https://nbviewer.org/github/landisland/MachineLearningAlgorithmsWithPlainPython/blob/master/SVM/svm_from_scratch.ipynb)
